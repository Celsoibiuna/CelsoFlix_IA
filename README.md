# Netflix Clone

Projeto estático em HTML, CSS e JavaScript.

## Estrutura do repositório

- `index.html` - página inicial de seleção de perfil
- `style.css` - estilos da página inicial
- `script.js` - lógica de tema e seleção de perfil
- `imagens/` - imagens de perfil e recursos do site
- `catalogo/` - área de catálogo
  - `catalogo.html` - página de catálogo
  - `catalogo.css` - estilos do catálogo
  - `js/` - scripts do catálogo

## Como usar

1. Abra `index.html` no navegador.
2. Selecione um perfil para salvar o nome e a imagem no `localStorage`.
3. O `catalogo.html` usa esses dados para mostrar o perfil ativo.

## Observações

- Não há dependências externas para instalar.
- O repositório está organizado para ser publicado diretamente no GitHub Pages.
